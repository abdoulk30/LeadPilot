# Roadmap

LeadPilot is an AI agent that consolidates outbound sales lead
tracking, contact history, and document handoffs across Google
Workspace and Slack — so reps stop losing leads to scattered
spreadsheets and duplicate outreach.

## Phase 1 (current build)

- One consolidated, prioritized view of every lead across all your
  intake sheets
- Automatic tracking of every call, text, and email touch, so reps
  never duplicate outreach
- Instant Slack alerts to the back office the moment a deal file is
  complete
- Built with prompt-injection resistance from day one — the agent
  treats all lead data as literal text, never as instructions

## Later (not yet scoped)

- Multi-org support
- Richer analytics
- CRM integrations beyond Google Sheets

## Notes

This roadmap is intentionally written in plain, outcome-focused
language — no tool names, no API details, no internal thresholds.
Given LeadPilot handles real financial documents for real sales
orgs, anything more specific stays in the private planning repo.
